# Noirven 诺梵 高奢 UI 借鉴清单

## 方法来源

本项目采用 `awesome-design-md` 的方法：把优秀网站的视觉系统拆成 Markdown 设计文件，让开发代理可以按统一规则生成 UI，而不是只靠模糊描述。

参考仓库：

- https://github.com/VoltAgent/awesome-design-md

可借鉴的结构：

- Visual Theme & Atmosphere
- Color Palette & Roles
- Typography Rules
- Component Stylings
- Layout Principles
- Depth & Elevation
- Do's and Don'ts
- Responsive Behavior
- Agent Prompt Guide

## 设计方法参考

### Bugatti

链接：

- https://raw.githubusercontent.com/VoltAgent/awesome-design-md/main/design-md/bugatti/DESIGN.md

可借鉴：

- 纯黑画布、白色文字、极少 UI 元素。
- 摄影、字体、品牌字标是主角。
- 大量留白制造稀缺感。
- 透明描边按钮比实心按钮更克制。

用于 Noirven：

- 首页 hero 用黑色背景和珠宝特写，而不是花哨背景。
- 只保留一个主 CTA：预约私人顾问。
- 产品模块用大图和少量文字，不做密集商品墙。

### Apple

链接：

- https://raw.githubusercontent.com/VoltAgent/awesome-design-md/main/design-md/apple/DESIGN.md

可借鉴：

- 产品像博物馆藏品一样被展示。
- UI 后退，让产品图说话。
- 每个区块只有一个视觉焦点。

用于 Noirven：

- 每件孤品需要独立展台式展示。
- 文案短，图片锐利，结构干净。
- 首页不要堆过多卖点。

### Tesla

链接：

- https://raw.githubusercontent.com/VoltAgent/awesome-design-md/main/design-md/tesla/DESIGN.md

可借鉴：

- 极简到接近没有界面。
- 全屏摄影承担主要情绪。
- 单一 CTA 色，不制造视觉噪音。

用于 Noirven：

- 顶部导航必须轻，不压住珠宝画面。
- 首屏只讲品牌名、品牌信条和定制入口。

### Linear

链接：

- https://raw.githubusercontent.com/VoltAgent/awesome-design-md/main/design-md/linear.app/DESIGN.md

可借鉴：

- 近黑色画布、细边界、分层暗色面板。
- 单一品牌色只用于焦点和行动。
- 信息复杂时仍能保持克制。

用于 Noirven：

- 定制流程、预约表单、宝石规格可以用暗色面板和细线组织。
- 不用阴影，靠层级色和 hairline 分隔。

## 高奢珠宝网站参考

### Cartier High Jewelry

链接：

- https://www.cartier.com/en-us/clp-high-jewellery.html
- https://www.cartier.com/en-us/high-jewelry/latest-collections/en-equilibre/

可借鉴：

- 用 `Latest Collections`、`Markers of Style`、`Living Legacy`、`Savoir-faire` 组织高珠内容。
- `En Équilibre` 强调 clean lines、positive/negative space、precision、nothing in excess，和 Noirven 的极简高奢非常契合。

用于 Noirven：

- 首页可设置：品牌传说、风格标记、孤品定制、工艺传承。
- 定制页面强调平衡、比例、留白、材质对照。

### Van Cleef & Arpels High Jewelry

链接：

- https://www.vancleefarpels.com/us/en/collections/high-jewelry.html

可借鉴：

- 高珠内容分为 Classic High Jewelry、Thematic collections、Signature collections、Heritage collection。
- 叙事维度很强：自然、文学、爱情、舞蹈、旅程、秘密。

用于 Noirven：

- Noirven 不只按戒指/项链分类，更应按故事分类：誓约、重生、家族印记、暗夜珍藏。
- 高定问卷要先问故事，再问材质。

### Graff High Jewellery

链接：

- https://www.graff.com/high-jewellery.html

可借鉴：

- 高珠内容按宝石组织：White Diamonds、Yellow Diamonds、Emeralds、Rubies、Sapphires、Rare Fancy Coloured Diamonds。
- 页面强调 `rare diamonds and gemstones`、`master artisans`、`unique creations`。

用于 Noirven：

- 建立按宝石进入的浏览方式：黑钻、白钻、祖母绿、蓝宝石、红宝石、珍珠母贝。
- 孤品页需要明确展示宝石规格、来源、切工和可定制方向。

### Boucheron High Jewelry

链接：

- https://www.boucheron.com/int_en/high-jewelry

可借鉴：

- 高珠分为 `Histoire de Style` 和 `Carte Blanche`。
- 一个强调档案和传承，一个强调自由创作。

用于 Noirven：

- 可以把产品体系分为两个入口：`Noir Archive` 经典暗夜风格，`Carte Noire` 完全私人自由委托。

### Tiffany High Jewelry

链接：

- https://www.tiffany.com/jewelry/shop/high-jewelry/diamonds/

可借鉴：

- 高珠页面结合品牌历史、最新高级珠宝系列、传奇宝石、专家联系入口。
- 明确提供 High Jewelry Expert 咨询方式。

用于 Noirven：

- 私人顾问入口必须清楚，不要只做普通联系表。
- 可设置 `Noirven Private Advisor` 作为核心转化路径。

### Chaumet High Jewelry

链接：

- https://www.chaumet.com/us_en/high-jewellery/collections/soir-de-fete
- https://www.chaumet.com/us_en/signature-collection-sentiments

可借鉴：

- 强调 Place Vendôme 工坊、手工传承、情感转化。
- `Sentiments` 以情感和人与人之间的纽带组织叙事。

用于 Noirven：

- 诺梵的“承诺/契约感”可以借鉴这种情感叙事，但视觉要更暗、更私密。
- 关于页面应强调“私人誓约被制作为孤品”。

### Chopard High Jewellery

链接：

- https://www.chopard.com/en-us/our-high-jewellery-universe.html
- https://www.chopard.com/en-us/high-jewellery-universe/precious-lace.html

可借鉴：

- 将高珠和工坊手艺、伦理采购、红毯场景绑定。
- 内容有明确主题系列，如 Red Carpet、Precious Lace。

用于 Noirven：

- 可加入材料伦理、宝石筛选、工匠手作过程。
- 不建议过度红毯化；Noirven 更适合私密 salon 方向。

### Harry Winston High Jewelry

链接：

- https://www.harrywinston.com/en/high-jewelry

可借鉴：

- 以系列入口组织高珠：New York、Winston Icons、The Incredibles、Legacy。
- 强调 one-of-a-kind、exceptional jewels、House legacy。

用于 Noirven：

- Noirven 可建立 `One-of-One` 孤品体系。
- 详情页要有“此件仅为一人而制”的唯一性表达。

## Noirven 最终 UI 方向

关键词：

- Dark private atelier
- Sacred solitaire
- One-of-one commission
- Minimal haute joaillerie
- Black lacquer, platinum, champagne gold

首页首屏建议：

- 背景：黑色全屏，中央或偏右一件锐利珠宝特写。
- 顶部：极简导航，居中 Noirven 诺梵。
- 主文案：`暗夜中的神圣孤品`
- 副文案：`为不可复制的誓约，定制唯一之物。`
- CTA：`预约私人顾问`

核心禁忌：

- 不做普通珠宝商城首页。
- 不做大面积金色背景。
- 不使用花哨渐变、闪光粒子、漂浮装饰。
- 不堆砌产品卡片。
- 不用“低价、热卖、新人优惠”等大众零售语言。
